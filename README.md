# URU Strength

Forge-themed strength tracker and 4-week program generator built for a Bane / Juggernaut style physique.

## v1 Features
- 4-week URU cycle
- Foundation / Strength / Power / Deload phases
- Bench, squat, and deadlift cornerstones
- Landmine thruster ladder
- Daily weight and workout log
- Infinity Forge final boss scorecards
- Local browser save

## Tech
- HTML
- CSS
- JavaScript
- localStorage
